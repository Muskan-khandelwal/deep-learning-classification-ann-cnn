# Deep Learning Image Classification using ANN and CNN

## Overview
This project demonstrates image classification using Artificial Neural Networks (ANN) and Convolutional Neural Networks (CNN).

The project compares the performance of ANN and CNN models using Deep Learning techniques.

---

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

---

## Concepts Covered
- Artificial Neural Networks
- Convolutional Neural Networks
- Deep Learning
- Image Classification
- Model Evaluation

---

## Dataset
Fashion MNIST Dataset

---

## Results
The models were evaluated based on classification accuracy.

CNN performed better than ANN for image classification tasks.
